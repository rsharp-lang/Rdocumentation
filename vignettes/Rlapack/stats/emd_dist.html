emd_dist {stats} R Documentation

Earth Mover's Distance

Description


Implementation of the Fast Earth Mover's Algorithm by Ofir Pele and Michael Werman.

Usage

emd_dist(x, y,
    bins = 10,
    extra.mass.penalty = -1);

Arguments

x

the sparse matrices being compared

y

the sparse matrices being compared

bins

[as integer]

extra.mass.penalty

penalty for extra mass. 0 for no penalty, -1 for the default, other positive values to specify the penalty; An extraMassPenalty of -1 means that the extra mass penalty is the maximum distance found between two features. [as double]

env

[as Environment]
