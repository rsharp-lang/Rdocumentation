svg {grDevices} R Documentation

Cairographics-based SVG, PDF and PostScript Graphics Devices

Description


Graphics devices for SVG, PDF and PostScript
graphics files using the cairo graphics API.

Usage

svg(
    image = NULL,
    file = NULL,
    ... = NULL);

Arguments

image

-

file

the name of the output file. The page number is substituted if a C integer format is included in the character string, as in the default. (Depending on the platform, the result must be less than PATH_MAX characters long, and may be truncated if not. See postscript for further details.) Tilde expansion is performed where supported by the platform.
