HtmlElement

Description

一个标签所标记的元素以及内部文本

在选择器里：

  • ID 和 类 选择器 区分 大小写

  • 标签选择器、属性选择器不区分大小写

    类选择器和 ID 选择器可能是区分大小写的。这取决于文档的语言。
    HTML 和 XHTML 将类和 ID 值定义为区分大小写，所以类和 ID
    值的大小写必须与文档中的相应值匹配。

    —— W3C


this class extends from InnerPlantText class.

Declare

            
# namespace Microsoft.VisualBasic.MIME.Html.Document
export class HtmlElement extends InnerPlantText {
   # 标签名
   TagName: string;
   # 标签的属性列表
   Attributes: ValueAttribute[];
   HtmlElements: InnerPlantText[];
   IsPlantText: boolean;
   # 唯一编号是区分大小写的
   id: string;
   name: string;
   class: string;
   OnlyInnerText: boolean;
   IsEmpty: boolean;
   InnerText: string;
}
