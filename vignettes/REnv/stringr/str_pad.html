str_pad {stringr} R Documentation

Pad A String.

Description

Usage

str_pad(string, width,
    side = left,
    pad = " ");

Arguments

string

A character vector.

width

Minimum width of padded strings.

side

Side on which padding character is added (left, right or both).

pad

Single padding character (default is a space).

Details

Value

this function returns data object of type string.
