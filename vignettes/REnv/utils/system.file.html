system.file {utils} R Documentation

Find Names of R System Files

Description


Finds the full file names of files in packages etc.

Usage

system.file(fileName,
    package = NULL,
    mustWork = FALSE);

Arguments

fileName

-

package

a character String With the name Of a Single package.
An Error occurs If more than one package name Is given.

env

-

Details

do not add extension suffix name for csv data set due to
the reason of csv file extension suffix will be removed
automatically in the progress of R compile the data file
into package file.

Value

this function returns data object of type any kind.

clr value class

  • any kind
